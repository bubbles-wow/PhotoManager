# PhotoManager  
一个图片管理程序，使用Java Swing实现

## 已实现功能
1. 目录树以及其他文件夹切换方式
2. 图片格式支持：`jpg`, `jpeg`, `png`, `bmp`, `gif`
3. 图片预览页面及逻辑（选中文件、右键菜单等）
4. 文件复制、粘贴、删除、单个文件重命名
5. 文件操作冲突提示
